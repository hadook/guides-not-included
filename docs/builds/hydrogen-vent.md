# Hydrogen vent taming


## Description

This design pumps out 130-140 degree hydrogen. Even if started in a vacuum, it won't overheat.


!!! abstract "Construction"

    **Atmo Sensor (upper left)**: Above 4000g  
    **Atmo Sensor (below gas pump)**: Above 1000g  
    **Thermo Sensor**: Below 135
    
    Use steel for the Gas Pump.


!!! tip

    Radiant liquid pipes help cool the steam turbine. You can also enclose the steam turbine and let the radiant pipes handle the cooling. And, of course, you can hook the steam turbine to your base's heavy wire network instead, if you want to harness the power for your base.
