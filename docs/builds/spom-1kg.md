# SPOM, Oxygen: 1kg/s


## Description

This is Francis John's take on the classic SPOM, modified to fit in a four-tile high area.


!!! abstract "Construction"

    **Atmo sensor, upper (hydrogen)**: above 750  
    **Atmo sensor, lower (oxygen)**: above 450

    Use gold (or better) for things can can overheat.  
    The upper section can be fully enclosed, the hydrogen cools the top level.


!!! example

    For an introduction to how the SPOM works, see: [Oxygen: To know the SPOM is to love the SPOM]().
