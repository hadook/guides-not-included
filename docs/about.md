# About


## Origin

I am not the author of these guides. I have built this website as an attempt to revive an old classic from the depths of web archive.

The content was originally built by some random Finn and published at [guidesnotincluded.com][original-site]. The original author has announced the website going down in this [reddit post](https://www.reddit.com/r/Oxygennotincluded/comments/1mj5k3f/about_guidesnotincluded/).

There had been many people offering to rebuild the site. As far as I know, none of those attemps have come to fruition, so here is mine.
